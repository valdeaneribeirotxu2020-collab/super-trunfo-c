# super-trunfo-c

Projeto desenvolvido em linguagem C para cadastro e exibição de duas cartas do jogo Super Trunfo.

## Funcionalidades
- Leitura dos dados de duas cartas
- Armazenamento em variáveis adequadas
- Exibição organizada das informações

## Dados das cartas
- Estado
- Código
- Nome da cidade
- População
- Área
- PIB
- Pontos turísticos

Projeto desenvolvido para fins acadêmicos.
